Mount is a very basic command that is used to mount filesystems like your cd,pendrives and other such external devices.
<blue>Usage:mount options file 
Examples:
1.This command <green> mount | column -t </green> can be used to display all the drives that you have presently mounted on the computer and their respective specs with the added benifit of nice formating.The pipe(|) is basically to pass the output of mount to the command column.
